## Setup Instructions

### 1. Create a `.env` file
Add the following environment variables to a `.env` file in the root directory:

```env
MONGO_URI=...
PORT=...
JWT_SECRET=...
NODE_ENV=...
CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...
```
### 2. Build the app
```
npm run build
```

### 3. Start the app
```
npm run start
```
